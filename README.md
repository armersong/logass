# logass

#### 介绍
日志过滤、处理工具。功能
- 日志过滤：去掉不需要关注的行
- 日志颜色：按照不同等级，渲染成不同的颜色

#### 软件架构
软件架构说明


#### 安装教程
cargo build --release

#### 使用说明
plugins 目录，保存插件对应配置。文件名是和插件同名。
例如：color(颜色过滤)插件，配置文件名为: color.json
